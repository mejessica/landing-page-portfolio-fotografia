# 🎨 Projeto Frontend — HTML & CSS

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte de um bloco acadêmico, com foco na construção de interfaces utilizando **HTML e CSS**.

O código foi pensado não apenas para funcionar, mas para seguir **princípios fundamentais de design**, garantindo uma experiência visual agradável, intuitiva e eficiente para o usuário.

---

## 🎯 Objetivo

Criar uma interface visualmente organizada e funcional, aplicando conceitos de design e boas práticas de desenvolvimento front-end.

---

## 🧠 Princípios de Design Aplicados

O projeto foi estruturado com base em princípios essenciais do design, como:

* **Hierarquia Visual** → organização dos elementos para guiar o olhar do usuário
* **Contraste** → destaque de informações importantes
* **Alinhamento** → criação de uma estrutura limpa e consistente
* **Proximidade** → agrupamento lógico de informações relacionadas
* **Repetição** → consistência visual em cores, tipografia e componentes
* **Simplicidade** → foco na clareza e usabilidade

---

## 💻 Tecnologias Utilizadas

* **HTML5** → estruturação semântica do conteúdo
* **CSS3** → estilização e layout da interface

---

## 🎨 Decisões de Design

* Uso de um **layout limpo e minimalista**, facilitando a navegação
* Escolha de cores pensadas para transmitir clareza e identidade visual
* Tipografia aplicada para melhorar a legibilidade
* Organização responsiva (quando aplicável)
